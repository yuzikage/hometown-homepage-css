# Visit Rajpipla - Hometown Homepage

A vibrant, responsive landing page dedicated to showcasing the culture and natural beauty of **Rajpipla**, a town in the Narmada district of Gujarat. This project was built to practice layout design, typography, and CSS transitions.

* **Interactive Activities:** A "Top Three" section featuring circular images that expand on hover.
* **Guide Profile Card:** A dedicated section for a local guide, featuring a scaling effect to draw user attention.
* **Elegant Typography:** Integrated with "Cormorant Garamond" from Google Fonts for a classic, sophisticated feel.
* **Pure CSS Interactivity:** No JavaScript is used; all animations and interactive states are handled via CSS `:hover` pseudo-classes.
